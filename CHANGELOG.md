# Changelog

所有值得注意的更改都将记录在此文件中。

## [0.2.0] - 2026-06-04

###新功能 ✨

- **Anthropic API 支持**：现在兼容Anthropic Messages API 格式
- **工具调用优化**：针对JSON partial_json 输出添加特殊处理逻辑
- **安全放行检查**：识别Markdown、数字等特殊场景，避免误截断

###改进 🚀

- **正则表达式增强**：扩展CJK 字符集支持范围
- **日志系统**：添加DEBUG 级别日志便于调试
- **文档完善**：新增中文README、详细使用指南和贡献文档

### Bug 修复 🐛

-修复流式响应中JSON 符号结尾的截断问题
-优化缓冲区策略，减少误截断风险

## [0.1.0] - 2026-06-03

###初始发布 🎉

- OpenAI Chat Completions API 支持
- OpenAI Completions API 支持
- CJK↔ASCII 空格自动修复
- Docker Compose 部署方案
-基础文档和测试套件

---

##版本命名规范

采用语义化版本号(SemVer)：`主版本。次版本.修订版`

- **主版本**：不兼容的API 变更
- **次版本**：功能性的向下兼容变更
- **修订版**：向下兼容的问题修正
